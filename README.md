# 🚀 Vivek Saini — Developer Portfolio

A modern, responsive personal portfolio website built with **React.js**, **Tailwind CSS**, and **Vite**. Showcasing my skills, experience, projects, and education with smooth animations, parallax tilt effects, and a polished dark-themed UI.

🔗 **Live Demo:** [vivek-saini-portfolio.vercel.app]((https://portpolio-xi-seven.vercel.app/))

## ✨ Features

- **Responsive Design** — Fully optimized for desktop, tablet, and mobile devices
- **Dark Theme** — Sleek dark UI with purple accent gradients and glassmorphism effects
- **Typing Animation** — Dynamic role display using React Typing Effect
- **Parallax Tilt Cards** — Interactive 3D tilt effects on profile image, experience, and education cards
- **Horizontal Scroll Sections** — Modern horizontal card carousels for Experience and Education
- **Skills Grid** — Categorized skill display (Frontend, Backend, Languages, Tools, Core Concepts, Coding Profiles)
- **Project Showcase** — Grid-based project cards with modal popups for detailed view
- **Contact Form** — Integrated with EmailJS for direct email communication
- **Social Links** — GitHub, LinkedIn, LeetCode, Codeforces, and CodeChef profiles
- **Smooth Scrolling** — Seamless navigation between sections
- **Custom Scrollbar** — Styled scrollbar matching the purple theme
- **Blur Blob Background** — Animated decorative background element

## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18.x
- **npm** ≥ 9.x

### Installation

```bash
# Clone the repository
git clone https://github.com/VivekSaini2005/Portfolio.git

# Navigate to project directory
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be running at `http://localhost:5173`

### Build for Production

```bash
npm run build
```
